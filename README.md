# Veri İletişimi Teknikleri - Soru Bankası

Veri İletişimi Teknikleri dersi için interaktif soru bankası uygulaması.

## Özellikler

- 110 soru içeren kapsamlı soru bankası
- Rastgele soru sıralaması (1. sorudan başlar)
- Otomatik cevap kontrolü
- Yanlış cevaplarda detaylı açıklama
- İstatistik takibi
- Son çözülen sorular listesi
- Responsive tasarım

## Teknolojiler

- HTML5
- CSS3
- Vanilla JavaScript

## Kurulum

1. Projeyi klonlayın veya indirin
2. `index.html` dosyasını tarayıcıda açın

## Vercel'e Deploy

```bash
# Vercel CLI ile
npm i -g vercel
vercel
```

Veya GitHub'a push edip Vercel web arayüzünden bağlayabilirsiniz.
